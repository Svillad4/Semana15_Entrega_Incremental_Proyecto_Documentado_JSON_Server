# Glosario rapido

**Documentacion tecnica:** explicacion escrita que permite entender, instalar, ejecutar y revisar un proyecto.

**README:** archivo principal de documentacion de un proyecto.

**Bitacora:** registro ordenado de pruebas, errores, correcciones y evidencias.

**Validacion:** proceso de comprobar que los datos o acciones cumplen reglas definidas.

**Correccion:** cambio aplicado para solucionar un error o mejorar un comportamiento.

**JSON Server:** herramienta que simula una base de datos local usando un archivo JSON.

**API:** forma en que el proyecto consulta, guarda o modifica datos.

**Endpoint:** URL especifica de una API, por ejemplo `/visitantes`.

**Dashboard:** panel visual que resume informacion importante.

**Entrega incremental:** version nueva del proyecto que agrega una mejora o funcionalidad respecto a una version anterior.

**Exposicion tecnica:** presentacion donde se explica el proyecto, su funcionamiento, pruebas y decisiones.
