# Entrega en Moodle

Antes de subir tu trabajo, comprime la carpeta completa del proyecto en formato `.zip`.

## Debe incluir

- Codigo fuente completo.
- `README_ENTREGA.md` diligenciado.
- `docs/bitacora-validacion.md` diligenciado.
- `docs/checklist-requisitos.md` diligenciado.
- `docs/guion-exposicion.md` diligenciado.
- `docs/decisiones-tecnicas.md` diligenciado.
- Carpeta `evidencias/` con capturas.

## Nombre sugerido del archivo

```text
Semana15_Entrega_Incremental_NombreApellido.zip
```

## Recomendacion

No elimines `package.json`, `db.json`, `src/`, `docs/` ni `evidencias/`.
