# Roles sugeridos para trabajo en equipo

Si trabajas en equipo, pueden organizarse asi:

| Rol | Responsabilidad |
|---|---|
| Coordinador | Organiza tareas y revisa que todos aporten. |
| Desarrollador | Ajusta codigo y valida funcionamiento. |
| Tester | Ejecuta pruebas y toma evidencias. |
| Documentador | Completa README, bitacora, checklist e informe. |
| Presentador | Prepara y lidera la exposicion tecnica. |

Un estudiante puede asumir mas de un rol si el equipo es pequeno.
